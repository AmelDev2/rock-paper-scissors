# Rock Paper Scissors

A simple command-line Rock Paper Scissors game built with Python.

## Features

- Play against the computer.
- Supports both short and full inputs:
  - `r` or `rock`
  - `p` or `paper`
  - `s` or `scissors`
- Input validation.
- Random computer choices.
- Win, lose, and tie detection.
- Play multiple rounds without restarting the program.

## How to Run

```bash
python rock_paper_scissors.py
```

## Technologies Used

- Python 3
- random module
